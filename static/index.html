Perguntas frequentes
Como registro um gasto pelo WhatsApp?+
Mande uma mensagem natural para o bot: "gastei 50 no mercado", "paguei 180 de luz". O contix interpreta e registra automaticamente.
Como funciona a cobrança recorrente?+
O Mercado Pago debita automaticamente todo mês na data da sua assinatura.
Como cancelar minha assinatura?+
Vá em Assinatura → Ações → Cancelar. Você mantém o acesso até o fim do período pago.
Meus dados aparecem para outros?+
Não. Cada conta é completamente isolada com autenticação JWT.
