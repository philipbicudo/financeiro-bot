Perguntas frequentes
Como registro um gasto pelo WhatsApp?+
Mande uma mensagem natural: "gastei 50 no mercado", "paguei 180 de luz". O contix interpreta e registra automaticamente.
Como cancelo minha assinatura?+
Vá em Assinatura → Ações → Cancelar. Você mantém o acesso até o fim do período pago.
Meus dados aparecem para outros usuários?+
Não. Cada conta é completamente isolada. Só você acessa suas transações.
Como funciona a cobrança recorrente?+
O Mercado Pago debita automaticamente todo mês. Você recebe confirmação por WhatsApp a cada pagamento.
Como baixar o recibo de pagamento?+
Vá em Assinatura → Ações → Baixar recibo. Um PDF com os detalhes da assinatura é gerado para impressão.

Termos de uso e Política de privacidade
Jan/2026

1. Aceitação dos termos

Ao utilizar o contix, você concorda com estes termos. O contix é um serviço de gestão financeira pessoal oferecido por Philip Bicudo.

2. Uso dos dados

Seus dados financeiros são armazenados de forma criptografada e nunca são compartilhados com terceiros. Não vendemos, alugamos ou transferimos seus dados pessoais.

3. Pagamentos e assinaturas

Os planos pagos são cobrados mensalmente via Mercado Pago. Você pode cancelar a qualquer momento mantendo acesso até o fim do período pago.

4. Segurança

Implementamos autenticação OTP, criptografia JWT e isolamento de dados por usuário.
